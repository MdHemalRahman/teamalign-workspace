# Core Features

EMP Open Source is organized around a small set of operational feature groups.

## Time & Attendance

These features record when work happens.

- Check-in
- Check-out
- Break tracking (lunch, prayer, and custom breaks)

## Work Tracking & Accountability

These features record what work is done.

- TO-DO management
- RDC task tracking
- Daily work summaries
- General activity and task tracker

EMP records work to reduce ambiguity, not to surveil employees.

## Leave & Compensation

These features record approved time away and salary-related adjustments.

- Leave requests and approvals
- Advance salary records
- Leave and attendance policies

## Notifications

EMP provides system-generated notifications only.

- Approval status updates
- Reminders
- Policy-related alerts

No real-time chat or external messaging integrations are included.

## Organization Structure

These features define how people are grouped.

- Team separation
- Basic role boundaries

## User & System

These features control identity and behavior.

- User profiles
- Application and user settings
